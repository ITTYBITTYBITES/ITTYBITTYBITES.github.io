# Two Second Witness

**Can You Trust What You Just Saw?**

A fast-paced visual memory and perception game where every scene lasts only two seconds. One detail changes. Your mind decides what it remembers.

---

## About This Repository

This repository hosts the **coming-soon splash landing page** for Two Second Witness — an upcoming indie memory and perception game.

It serves as the placeholder marketing site prior to the launch of the full playable demo.

## Structure

```
/website/
├── index.html          — Splash landing page
├── css/
│   └── main.css        — All styles (no frameworks)
├── js/
│   └── main.js         — Minimal interactions (no dependencies)
└── assets/
    └── images/         — Images and favicon (placeholder)
README.md
```

## Features

- Pure HTML, CSS, and vanilla JavaScript — zero dependencies
- Cinematic dark theme with neon accent glow
- Animated gradient background with noise texture overlay
- Responsive design for mobile and desktop
- Full SEO meta tags (Open Graph, Twitter Card)
- Respects `prefers-reduced-motion` accessibility setting
- Loads in under 1 second

## Deployment

This site is designed for **GitHub Pages** static hosting. To deploy:

1. Push to the `main` branch
2. Enable GitHub Pages in repository settings (source: root or `/website` folder)
3. The splash page will be live at your GitHub Pages URL

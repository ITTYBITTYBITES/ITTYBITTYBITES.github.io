Frequently Asked Questions

Everything you need to know about the two-second perception mechanic, upcoming web playable demo, and project roadmap.

General Questions

What is Two Second Witness?

Two Second Witness is an indie visual perception and working memory game. Players are exposed to high-detail 3D and 2D visual scenes for precisely two seconds before the screen blacks out. You must then recall fine details or spot micro-anomalies under intense pressure.

Why exactly two seconds?

Cognitive research shows that human iconic memory processes immediate visual snapshots within 1 to 2 seconds before encoding into short-term working memory. Two seconds is the exact cognitive threshold where casual glancing fails and active, intentional visual filtering is required.

When will the playable demo be available?

We are currently completing Phase 2 (static architecture and content scaffolding). Phase 3 will introduce our interactive HTML5/WebGL demo directly into this website. Stay tuned to our development updates on the Dev Blog.

Technical & Platforms

What platforms will Two Second Witness support?

The interactive web demo is built natively for desktop browsers and mobile web via GitHub Pages. Future standalone releases are planned for PC (Steam) and mobile touchscreen devices (iOS & Android).

Does the site use any external JavaScript frameworks?

No. In alignment with our strict performance targets (<1.5s load times), this website is built using 100% pure semantic HTML5, modern CSS variables and flex/grid systems, and lightweight vanilla JavaScript. Zero frameworks or external libraries are loaded.

How can creators or press contact the team?

Journalists, content creators, and streamers can find our complete brand guidelines, fact sheets, and media assets on our Press Kit page.
